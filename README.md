## Run Locally

#### Clone the project

```bash
  git clone https://github.com/viniciusmb17/03-dt-money
```

#### Go to the project directory

```bash
  cd my-project
```

#### Install dependencies

```bash
  yarn add
```
or
```bash
  npm install
```

#### Start development environment

#### Front-End:
```bash
  yarn dev
```
#### Json-Server:
```bash
  yarn dev:server
```

or

#### Front-End:
```bash
  npm run dev
```
#### Json-Server:
```bash
  npm run dev:server
```
